Maze Solver Encryption System

The Maze Solver Encryption System is a cryptographic technique that uses an algorithm inspired by maze solving to generate encryption keys. The system maps out a maze on a grid, and the pathfinding algorithm generates a unique key based on the steps taken to navigate the maze. This key is then used to encrypt and decrypt messages using XOR-based operations, ensuring secure communication.

Smart Inventory Management System

The Smart Inventory Management System is a Python-based application designed to optimize inventory tracking and management for businesses. By using real-time data analytics, it predicts stock shortages, automates reordering, and tracks product sales trends. The system also includes an integrated loyalty program that rewards frequent suppliers and helps businesses streamline their supply chain and reduce overhead costs.
